# Shopping cart app

Simple app for android.

---

## Key Features

1. **Shopping cart:**
   - Add product.
   - Edit product.
   - Remove product.
   - Check product as bought

2. **Account system:**
   - Manage account.
   - Accounts have they private shopping carts.

3. **Preferences:**
   - Manage text size
   - Manage currency
   - Manage text color

---

## Technologies Used in the Project

- **App:** Android Studio
- **Language:** Kotlin
- **Database:** Firebase
- **Version Control:** Git
